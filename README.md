# Лендинг "Подготовка к ЕГЭ по информатике"

## Тема проекта
Сайт для онлайн-подготовки к ЕГЭ по информатике.

## Использованный фреймворк
Bootstrap 5

## Структура сайта
- Хедер с навигацией
- Hero-секция с призывом
- О проекте
- Преимущества (карточки)
- Программа курсов
- Отзывы
- Футер с контактами
