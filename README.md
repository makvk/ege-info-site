# Лендинг "Подготовка к ЕГЭ по информатике"

## Тема проекта
Сайт для онлайн-подготовки к ЕГЭ по информатике.

## Использованный фреймворк
Bootstrap 5

## Страницы сайта
- Главная (index.html)
- FAQ с аккордеоном (faq.html)
- Контакты с формой обратной связи (contacts.html)
- Страница 404 (404.html)

## Функционал
- Адаптивная вёрстка
- Форма обратной связи с валидацией
- Аккордеон для часто задаваемых вопросов
- Кастомная страница ошибки 404
- Единая навигация на всех страницах
